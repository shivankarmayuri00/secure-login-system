# 🔐 Secure Authentication System

## 📌 Project Description

This is a Secure Login and Registration System built using Flask (Python).
It allows users to create an account and login securely with password hashing.

---

## 🚀 Features

* User Registration
* User Login
* Password Hashing (bcrypt)
* SQLite Database
* Modern UI (Glassmorphism)
* Animated Background
* 🦋 Butterfly Animation with Wing Flapping
* Loader Animation

---

## 🛠️ Technologies Used

* Python (Flask)
* SQLite
* HTML, CSS
* bcrypt

---

## ▶️ How to Run

1. Install dependencies:

```
pip install Flask bcrypt
```

2. Run the app:

```
python app.py
```

3. Open in browser:

```
http://127.0.0.1:5000
```

---

## 📸 Output

Modern login system with animations and secure authentication.
